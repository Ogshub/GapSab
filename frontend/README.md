# GabShup — Frontend

The frontend for the GabShup real-time chat application, built with **React 19**, **Vite**, **Tailwind CSS**, and **Socket.IO Client**.

For full project documentation, setup instructions, and API details, see the **[main README](../README.md)** in the project root.

## Quick Start

```bash
npm install
npm run dev
```

Opens at `http://localhost:5173`
